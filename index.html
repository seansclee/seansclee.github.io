More about me

I'm an experienced software developer specializing in front end development. I've mostly built web apps in React, but throughout my career I've had the opportunity to "wear many hats" and work with a variety of technologies and tools. Notable areas of expertise in front end development include: architecting complex single page layouts, designing and developing responsive components, analyzing and optimizing user flows, gauging and applying web accessibility standards, and ensuring cross-browser compatibility. Lately, I've been working on projects concerning web security, Apache server configurations, SEO, and page speed optimizations.

Currently, I work at Thorium Digital, where I get to collaborate with a team of driven and talented individuals to create really neat web experiences for globally recognized brands.
